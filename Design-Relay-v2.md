## Design Philosophy

**Inspiration**: OpenAI, Vercel, Linear (light mode)
**Core principle**: Typography-first, whitespace-heavy, zero gimmicks

### Visual Identity

**Colors**:
- Background: `#FAFAFA` (off-white, not pure white)
- Surface: `#FFFFFF` (cards)
- Text Primary: `#171717` (near-black)
- Text Secondary: `#737373` (muted gray)
- Text Tertiary: `#A3A3A3` (light gray)
- Accent: `#000000` (pure black for CTAs)
- Border: `#E5E5E5` (subtle)
- Border Hover: `#D4D4D4`

**Typography**:
- Primary: `Geist` (Vercel's font) or system fallback
- Weights: 400 (regular), 500 (medium), 600 (semibold)
- Headings: Tight letter-spacing (-0.02em), leading 1.1
- Body: Relaxed leading (1.6)

**Spacing**:
- Generous whitespace
- Section padding: 160px vertical
- Container max-width: 720px (reading width, not full-width)

**Elements**:
- No gradients
- No shadows (or extremely subtle)
- No decorative blobs/shapes
- 1px borders only
- Border radius: 6px (subtle)

## Sections

### 1. Navigation

Minimal top bar:
- Logo left
- Links center (Products, Pricing, Docs)
- Login + Get Started right

### 2. Hero

Centered, narrow container:
- Small label above: "AI Agent Builder"
- Large headline: "Build agents that work for you"
- Subheadline: "Create, train, and deploy AI agents in minutes. No code required."
- Large input field (prominent)
- No decorative elements

### 3. How It Works

Three items in a row:
- Number + Title + Description
- No icons, no cards
- Just text

### 4. Agent Types

List format, not cards:
- Agent name + brief description
- Horizontal line separators

### 5. Integrations

Simple grid of logos/names
- No colored backgrounds
- Just text or monochrome icons

### 6. Pricing

Three columns:
- Clean, minimal
- Black CTA button
- No highlighted "Pro" card with different colors

### 7. FAQ

Accordion style:
- Question + answer
- Clean expand/collapse

### 8. Footer

Minimal:
- Logo
- Links
- Copyright
